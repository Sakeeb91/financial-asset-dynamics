Stochastic Differential Equations

📈 Geometric Brownian Motion

Classical model for stock price dynamics with constant drift and volatility parameters. Supports exact analytical solutions and efficient Monte Carlo simulation.

dS = μS dt + σS dW

🌊 Heston Stochastic Volatility

Two-factor model capturing volatility clustering and correlation between price and volatility. Implements Euler-Maruyama scheme with variance floor for numerical stability.

dS = μS dt + √v S dW₁
dv = κ(θ-v) dt + ξ√v dW₂

⚡ Jump-Diffusion Models

Extensions of GBM incorporating discontinuous price movements via Poisson processes. Models market crashes and earnings announcements with configurable jump intensities.

dS = μS dt + σS dW + S dJ

Code Examples

Portfolio Optimization
from sde_asset_modeling.portfolio.optimizer import PortfolioOptimizer # Initialize optimizer with returns data optimizer = PortfolioOptimizer(returns_data) # Maximum Sharpe ratio portfolio max_sharpe = optimizer.maximum_sharpe_optimization() print(f"Optimal weights: {max_sharpe['weights']}") print(f"Sharpe ratio: {max_sharpe['sharpe_ratio']:.3f}") # Generate efficient frontier frontier = optimizer.efficient_frontier(n_points=100)
Risk Analysis
from sde_asset_modeling.portfolio.risk_metrics import RiskMetrics # Initialize risk calculator risk_calc = RiskMetrics(returns_data) # Value at Risk calculations var_95 = risk_calc.value_at_risk(confidence_level=0.05) cvar_95 = risk_calc.conditional_value_at_risk(0.05) # Comprehensive risk report risk_report = risk_calc.generate_risk_report()
Options Pricing
from sde_asset_modeling.options.black_scholes import BlackScholesModel # Create Black-Scholes model bs = BlackScholesModel( spot_price=100, strike_price=105, time_to_expiry=0.25, risk_free_rate=0.05, volatility=0.2 ) # Calculate prices and Greeks call_price = bs.call_price() greeks = bs.all_greeks('call') print(f"Delta: {greeks['delta']:.3f}")
Market Regimes
from sde_asset_modeling.analytics.regime_detection import RegimeDetector # Initialize regime detector detector = RegimeDetector(returns_data.iloc[:, 0]) # Detect volatility regimes vol_regimes = detector.volatility_clustering_regimes() # Gaussian mixture model regimes gmm_regimes = detector.gaussian_mixture_regimes(n_regimes=3)
